#Pawsitive

This repository contains the source for the Pawsitive static site served as a GitHub Pages site.

Quick start
- Serve locally: `python3 -m http.server 8765`
- Open: http://localhost:8765 in a browser of your choice

What’s in this repo
- `index.html` — site entry
- `js/` — JavaScript sources (gallery, dogs, etc.)
- `styles/` — CSS design files
- `public/` and `dogs/` — static assets and images

Documentation
- See the documentation folder for additional notes: [documentation](documentation/)

Deployment
- This is a static site; deploy by pushing to the repository used for GitHub Pages or serving the built files from a web server.
